# packages/numasec

Core numasec application package: TUI, server, agents, tools, operations, evidence, replay, and cyber kernel integration.

Start at the root [README](../../README.md) for product docs and release notes.

Package-local commands:

```bash
bun test --timeout 30000
bun run build
bun run bench:cyber --domain appsec
bun run bench:cyber --domain pentest
```
